# Straddie Maker-Space Lab

A public, multi-page Strange But True family site for a proposed light-industrial maker-space in Dunwich / Goompi on Minjerribah / North Stradbroke Island.

The site keeps the main public story practical:

- making useful things
- fixing and right-to-repair
- compact tool-shed design
- tool sharing and Markdown form builders
- recycling and upcycling
- a lawful tip-loop / pre-tip diversion sister-site concept
- a concrete batch-plant sister-site concept
- local sand and material literacy
- Ready S.E.T. Co-op pathways
- hyperlocal media and digital noticeboard links
- future-facing but bounded Sandworm / subterranean systems research
